Xero — Not Connected

Connect Xero to see invoices sent, payments received, outstanding amounts, and get answers like "Did my invoice get paid?" in the Family Chat.

Setup steps:
1. Go to developer.xero.com → My Apps → New App
2. Set redirect URI: your-app-url/xero/callback
3. Add Client ID + Secret to Coolify env vars:
  XERO_CLIENT_ID and XERO_CLIENT_SECRET
4. Restart and connect below

Setup Guides
Google Calendar:
1. Go to console.cloud.google.com → New Project
2. Enable Google Calendar API
3. Create OAuth 2.0 credentials, add redirect URI: your-url/google/callback
4. Add GOOGLE_CLIENT_ID and GOOGLE_CLIENT_SECRET to Coolify env vars

Xero:
1. Go to developer.xero.com → My Apps → New App
2. Add redirect URI: your-url/xero/callback
3. Add XERO_CLIENT_ID and XERO_CLIENT_SECRET to Coolify env vars

Note: The Family Chat already has full context about your property, birthdays, and goals — it works without these integrations connected.
